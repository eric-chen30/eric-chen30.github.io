# vscode 常用快捷键

  • 向上 / 下复制当前行

alt + shift + ⬆/⬇

  • 逐个选择文本

CTRL + shift + ⬅/➡

  • 批量替换匹配文本

CTRL + F2

  • 将当前行代码上 / 下移

alt + ⬆/⬇

  • 删除当前行

CTRL + x

  • 多行同时编辑

  1. CTRL + alt + ⬆/⬇

  2. alt + 鼠标选中特定位置

  • 快速定位文件首位行

CTRL + home/end

  • 删除上一个词

CTRL + backspace

  • 选择光标左 / 右侧全部内容

CTRL + shift + home/end

  • 按名称搜索文件

CTRL + p
